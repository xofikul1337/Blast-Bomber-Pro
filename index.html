ব্লাস্ট বোম্বার বটের বডি যুক্ত করার নিয়ম:

আপনার ভেরিয়েবল হলো: number

পয়েন্ট নাম্বার ১: number ভেরিয়েবল কি?

আপনি যখন বটে মেসেজ বোম্বিং করার সময়, বট আপনার কাছে আপনার ভিক্টিম এর ১১ ডিজিটের নাম্বার চায়, তখন আপনি যে নাম্বার টা ইনপুট দিবেন, সেটা number ভেরিয়েবল এর মধ্যে থাকবে। এজন্য যেকোনো এপিআই এর বডি বটে সাবমিট করার আগে সেখানে ১১ ডিজিট এর নাম্বার কেটে দিয়ে number লেখা টা বসিয়ে দিতে হবে। মূলত এটাই হলো number ভেরিয়েবল।
Example: "number" → number

খেয়াল করুন:

  • এপিআই এর বডির ভিতর যেখানেই আপনি নাম্বার বসাবেন, সেই নাম্বারের সামনে পিছনে কোনো Inverted কমা রাখবেন না।
  • ভুল লেখা: "number"
  • সঠিক লেখা: number

যদি আপনার বের করা এপিআই এর বডির ভিতর ১১ ডিজিটের নাম্বারের আগে +88 বা 88 লেখা থাকে, তাহলে ১১ ডিজিটের নাম্বার কেটে দিয়ে সেখান প্লাস দিয়ে নাম্বার ভেরিয়েবল লিখতে হবে।

যেমনঃ +number

উধাহরণ ১:

এখন, ধরুন আপনার এপিআই বডি এই রকম:

{"email": "example@gmail.com", "phone_number": "8801945284114"}

এখানে আপনাকে ১১ ডিজিটের মোবাইল নাম্বারটি কেটে দিতে হবে। অর্থাৎ, 0 থেকে বাকি নাম্বার গুলো কেটে দিতে হবে। তখন লেখাটি হবে এরকম:

{"email": "example@gmail.com", "phone_number": "88" +number}

এবং বটে আপনি যখন নাম্বার ইনপুট দিবেন তখন আমরা 01 থেকে শুরু করবো। এজন্য 0 এর আগে যেটা থাকবে সেটাকে ডাবল কোটেশন বা ইনভারটেড কমার মধ্যে রেখে, তার সাথে +number লিখে দেবো।

উধাহরন ২:

যদি আপনার বডির নাম্বার এর জায়গায় আগে পরে কোনো 88 বা +88 না থাকে, তাহলে সরাসরি আপনার বডিতে থাকা নাম্বার ও ইনভারটেড কমা কেটে দিয়ে number লিখে দিবেন।

{"phone_number": "01844445862"}

এখানে লেখাটি হবে:

{"phone_number": number}

আরো একটি উদাহরণ:

যদি আপনার এপিআই বডি এই রকম হয়:

{"phone_number": "+8801924981783", "sent": "success", "result": "true"}

তাহলে আপনি লিখবেন:

{"phone_number": "+88" +number, "sent": "success", "result": "true"}

মুল কথা:

আমরা বডিতে যা আছে তা না দেখে, শুধু নাম্বার এর জায়গায় যদি সরাসরি ১১ ডিজিটের নাম্বার দেয়া থাকে তাহলে ইনভারটেড কমা সহ নাম্বার কেটে দিয়ে সরাসরি number লিখে দিবো। এবং যদি ১১ ডিজিটের নাম্বার এর আগে পরে কোনো অতিরিক্ত অংশ থাকে তাহলে সেটা ইনভারটেড কমার মধ্যে রেখে তার সাথে +number লিখে দিবো। এভাবে আপনি যেকোনো এপিআই এর বডি লিখতে পারবেন।

মূল বিষয়:

  • API বডির ভিতরে নাম্বারের জায়গায় number লিখে দিবেন।
  • যদি নাম্বারের আগে কোন অতিরিক্ত অংশ থাকে (যেমন +88), তাহলে সেটা ইনভারটেড কমাতে রেখে +number লিখবেন।
  • যদি সরাসরি ১১ ডিজিটের নাম্বার থাকে, তাহলে ইনভারটেড কমা বাদ দিয়ে শুধু number লিখবেন।
